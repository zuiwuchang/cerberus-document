Cerberus 工具箱

cerberus 工具箱，是我在使用android時爲其開發的各種工具集合，用來使生活或工作變得更加舒暢高效。 我發現其中一些功能可能對別人也會有所幫助，所以決定將其整合發佈以供大家分享使用。

工具箱主要分爲私有功能和公共功能，大部分功能都是公共的可以供任何人免費使用，私有功能則是我用來管理自己的私人系統的大家可以忽略。

公共功能列表
  • http 雲盤 服務器 - (將手機作爲 雲盤服務器 提供檔案上傳 分享)
  • 條碼
  • 草榴社區
  • 編碼工具
  • ssh工具箱
  • ip工具箱
計劃添加功能
  • ssh 客戶端
  • 支持 德語 日語
http 雲盤 服務器
  • 將手機作爲一個 雲盤服務器 提供 http 服務
  • 支持 權限控制 保證上傳安全
  • http 端口依然支持復用 保證 路由不同即可
  • 檔案下載 支持 標準的 http range 可以分塊下載 恢復異常下載
  • 檔案上傳 使用 websocket + crc32 私有協議上傳 支持 異常恢復
  • 雲盤上的 視頻 音頻 支持在線 觀看
  • 雲盤上的 小檔案(小於 10 MB) 小圖片(小於 10 MB) 支持 在線查看
創建 qrcode
此功能可以將用戶輸入的一段字符串轉換爲qrcode條碼，並支持保存到手機或分享到社群。
識別 qrcode barcode
識別手機照相機或保存的圖像中的二維條碼。
草榴社區
  • 服務器 每天定時更新 草榴社區 電影資源 並創建索引
  • 爲客戶端 提供通過名稱 索引查詢 電影的服務
    • 編碼工具
    編碼工具 用於輸入一種格式的數據 自動將其轉換爲其它格式的數據 比如想要輸入 utf8文本 來獲取 它的 hex 值
  • 支持 輸入輸出 utf8 文本
  • 支持 輸入輸出 hex 形式的 二進制數據
  • 支持 輸入輸出 字符數組 形式的 二進制數據
  • 支持 輸入輸出 01 形式的 二進制數據
    • 數值轉換
    數值轉換 支持 將一個 int64 範圍內的 數據 在 多種進制中轉換
  • 支持 輸入輸出 十進制
  • 支持 輸入輸出 十六進制
  • 支持 輸入輸出 二進制
    • 密鑰管理
    • 使用 rsa 算法 生成密鑰
    • 刪除密鑰
    • 導出密鑰
    • 導入密鑰
    用戶管理
    • 存儲一個用戶以供ssh協議使用
    • 刪除用戶
    • 導出用戶
    • 導入用戶
    ssh本地端口轉發
    使用ssh協議的本地端口轉發，將一個手機端口映射到ssh服務器上指定地址。
    sftp 客戶端
  • sftp 客戶端 連接 sftp 服務器
  • 支持上傳 檔案/檔案夾
  • 支持下載 檔案/檔案夾
  • 支持 檔案/檔案夾 改名 刪除 mkdir
    • tcp轉發
    • 支持 tcp tls-tcp ws wss 協議
    • 將本地的一個tcp數據流 轉發到遠端
    • 比如在本地監聽一個端口 將此端口的數據轉發到 服務器上的 websocket 接口 會很有趣
    • 或者在本地監聽一個 websokcet服務 將收到的數據轉發到服務器上的 tcp 接口 會很有趣
    • websocket 必須使用 binary frame
    • websocket服務器支持 端口復用 只要 路由獨立就行 但 ws 和 wss 不可復用同一端口
    tcp 客戶端
    • 支持 tcp tls-tcp ws wss 協議
    • 以測試tcp爲目的 創建一個到服務器連接的 客戶端
    • 支持發送 utf8 文本 或 hex 二進制數據 到服務器
    • 接收服務器傳回的數據 以 utf8文本 或 hex 視圖顯示
    tcp 服務器
    • 支持 tcp tls-tcp ws wss 協議
    • 以測試tcp爲目的 創建一個服務器 供 客戶端連接
    • 支持發送 utf8 文本 或 hex 二進制數據 到服務器
    • 接收服務器傳回的數據 以 utf8文本 或 hex 視圖顯示
