Cerberus 工具箱

cerberus 工具箱，是我在使用android時爲其開發的各種工具集合，用來使生活或工作變得更加舒暢高效。 我發現其中一些功能可能對別人也會有所幫助，所以決定將其整合發佈以供大家分享使用。

工具箱主要分爲私有功能和公共功能，大部分功能都是公共的可以供任何人免費使用，私有功能則是我用來管理自己的私人系統的大家可以忽略。

公共功能列表
  • 條碼
  • ssh工具箱
  • ip工具箱
計劃添加功能
  • tcp 服務器/客戶端 測試工具(實施中)
  • sftp/ftp 客戶端
  • ssh 客戶端
  • 支持 英語 德語 日語
創建 qrcode
此功能可以將用戶輸入的一段字符串轉換爲qrcode條碼，並支持保存到手機或分享到社群。
識別 qrcode barcode
識別手機照相機或保存的圖像中的二維條碼。
密鑰管理
  • 使用 rsa 算法 生成密鑰
  • 刪除密鑰
  • 導出密鑰
  • 導入密鑰
用戶管理
  • 存儲一個用戶以供ssh協議使用
  • 刪除用戶
  • 導出用戶
  • 導入用戶
ssh本地端口轉發
使用ssh協議的本地端口轉發，將一個手機端口映射到ssh服務器上指定地址。
tcp轉發
  • 支持 tcp tls-tcp ws wss 協議
  • 將本地的一個tcp數據流 轉發到遠端
  • 比如在本地監聽一個端口 將此端口的數據轉發到 服務器上的 websocket 接口 會很有趣
  • 或者在本地監聽一個 websokcet服務 將收到的數據轉發到服務器上的 tcp 接口 會很有趣
  • websocket 必須使用 binary frame
  • websocket服務器支持 端口復用 只要 路由獨立就行 但 ws 和 wss 不可復用同一端口
